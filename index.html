About Me

My first exposure to coding came at 13, when I bought an old C primer at a used book sale. Quickly realizing the book was a bit over my head, I moved on to more manageable projects, like making webpages with HTML and CSS, writing text-based adventures in BASIC on my TI-84, and remaking Space Invaders with my cousin using the GameMaker framework.

While at Harvard, I dove into coding in CS50: Introduction to Computer Science. Besides finally learning how to code in C, I got my first taste of scripting languages, using PHP and Javascript, as well as a MySQL database, to build a site to solve a pressing on-campus problem: organizing the a cappella auditions process. I became a peer tutor for the course, helping other students work through projects like building a web server and integrating with the Google Maps API.
